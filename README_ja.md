# spre

**monome nornsのためのジェネレーティブ・シンセサイザー。**

spreは、12音ポリフォニーのSuperColliderシンセエンジンに、5種類のメロディ生成、8種類のポリフォニック・ボイシング、調整可能なコード・ストラム、スケール・クオンタイズ、MIDI／grid演奏、2方式のルーパーを組み合わせた楽器です。単体で自動演奏するほか、外部MIDIキーボードや16×8 gridからスケールに沿って演奏できます。

澄んだ共鳴音から、飽和したざらつき、古いテープのような質感まで変化します。演奏を止めずにメロディ、ハーモニー、音色を作り替えられます。

> [English manual](README.md)

## 必要なもの

- monome norns
- 16×8 grid：任意
- USB MIDIキーボード／コントローラー：任意

spreには専用のSuperColliderエンジンが含まれます。
初回インストール後はnornsの電源を完全に切り、もう一度電源を入れてください。ソフトウェア上の再起動だけでは不十分な場合があります。

プロジェクトの構成

```text
spre/
├── spre.lua
├── README.md
├── README_ja.md
└── lib/
    └── Engine_Spre.sc
```

## インストール

MaidenのREPLで実行します。

```text
;install https://github.com/puyoma/spre
```

インストール後はnornsの電源を完全に切り、もう一度電源を入れてから、SELECTで`spre`を起動してください。
ソフトウェア上の再起動だけを行うと、完全な電源OFF／ONまで`AUDIO SYSTEM FAIL`が表示される場合があります。

## クイックスタート

spreは起動すると**AUTO**モードになり、自動演奏を始めます。

- **E2**でパラメータを選択します。
- **E3**で選択中の値を変更します。
- FILTER選択中は**E1**でAIR／GLASS／AMBER／WOODを切り替えます。
- **K3**でMEL／POLY／STRUMページを開きます。
- AUTOの新しい発音を止めるには**FLOCK**を0にします。

最初はnorns本体の出力／ヘッドホンレベルを低めにしてください。サチュレーション、レゾナンス、コード、FLOCKとRAINの高い設定を組み合わせると、音量が急に変化する場合があります。

## norns本体の操作

特記がない限り、以下はメインのパラメータページでの操作です。

| 操作 | 機能 |
|---|---|
| E1 | FILTER選択中：AIR／GLASS／AMBER／WOODを切り替える |
| E2 | パラメータを選択する |
| E3 | 選択中のパラメータを変更する |
| K2 | 次のパラメータ行へ移動する |
| K3 | パラメータページとMEL／POLY／STRUMページを切り替える |
| K1＋E1 | パラメータ、MEL／POLY／STRUM、ビジュアライザーの各ページを切り替える |
| K1＋K2 | オクターブを下げる |
| K1＋K3 | オクターブを上げる |
| K2＋K3 | 選択中のオーディオループ、またはノートループのスロット1を消去する |

**LOOP**を選択しているとき、K2は次の行への移動ではなくルーパー操作になります。

MEL／POLY／STRUMページでは以下のように操作します。

| 操作 | 機能 |
|---|---|
| E2またはK2 | MEL、POLY、STRUMのどれを編集するか選ぶ |
| E3 | メロディ方式、ボイシングを選ぶ、またはストラム量を調整する |
| K3 | パラメータページへ戻る |

ビジュアライザーページには雨の路地裏のアニメーションが表示され、専用の編集操作はありません。K1＋E1で別のページへ移動します。

## モード

### AUTO

選択したROOTとSCALEから、内部ジェネレーターが音を生成します。

- **FLOCK**：発音イベントの頻度。0にすると新しい音が止まります。
- **RAIN**：生成されたイベントを休符にする確率。0ではすべて発音し、最大ではすべて休符になります。
- **ATK／DEC**：エンベロープの形を調整します。
- MELODY：音程がどのように動くかを選びます。
- POLY：各音をどのような和音で鳴らすかを選びます。
- STRUM：生成されたコードの発音タイミングを散らし、人が弾いたような揺らぎを加えます。

### MIDI

入力されたMIDIノートをROOTとSCALEへクオンタイズし、ADSRエンベロープで演奏します。パラメータページではATTACK、DECAY、SUSTAIN、RELEASEを調整できます。接続したgridは、MIDIモードのままでもスケール鍵盤として演奏できます。

### GRID

接続した16×8 gridがスケール配列の鍵盤になります。パッドを押している間ノートが鳴り、MIDIモードと同じADSRエンベロープを使用します。

## メロディ生成

| 名前 | 動き |
|---|---|
| RAND | 全体から均等にランダム選択。ときどき1オクターブ上へ跳ぶ |
| GAUSS | 音域の中央付近が出やすいガウス分布によるランダム移動 |
| MRKV | 直前の音の近くへ動きやすいマルコフ的な移動 |
| ORBIT | コードトーンへ引き寄せられ、アプローチや時折の跳躍を含む移動 |
| FOLD | 4音のモチーフが徐々に上昇と下降を繰り返す |

## ポリフォニック・ボイシング

音程は、選択中のスケールの度数に基づきます。

| 名前 | ボイシング |
|---|---|
| MONO | 単音 |
| OCT | 元の音＋1オクターブ上 |
| 5TH | 元の音＋スケール4度上 |
| TRIAD | 元の音＋スケール2度上＋4度上 |
| 7TH | TRIAD＋スケール6度上 |
| RAND | ランダムに選んだ上方のスケール音を1〜2音追加 |
| ADD4 | 元の音＋スケール2度上＋3度上 |
| JAZZ | 元の音＋スケール2度上＋6度上 |

## ストラム

STRUMは、AUTOモードで生成されるコード内の発音タイミングを調整します。0ではすべての音が同時に鳴ります。値を上げると、わずかなランダムさを含めて最大約130 msまで発音が散らされ、ストラムや人が弾いたような感触になります。MONOの単音には影響しません。

## 音色パラメータ

| パラメータ | 内容 |
|---|---|
| ATK | アタック |
| DEC | ディケイ |
| FM | ピッチ・モジュレーションの深さ |
| BRIT | 音色の明るさ |
| SAT | サチュレーションとドライブ |
| TAPE | サンプルレート低下、テープノイズ |
| FILTER | E3でカットオフ／レゾナンスを調整。左側は暗く共鳴が強く、右側は明るく開いた音。E1でAIR／GLASS／AMBER／WOODを選択 |
| DUST | 高域のノイズ成分 |
| SPREAD | ランダムなステレオ配置 |
| INTONE | チューニングの色付け。中央は平均律、中央から離すと音程ごとのオフセットが加わる |
| SUSTAIN | MIDI／GRIDでのADSRサステイン |
| RELEASE | MIDI／GRIDでのADSRリリース |

### フィルタータイプ

FILTERを選択した状態でE1を回すか、PARAMSのFILTER TYPEから切り替えます。変更は次に発音する音から適用され、すでに鳴っている音のフィルターは変わりません。

| タイプ | 特徴 |
|---|---|
| AIR | 明るく開いた12 dBローパス。レゾナンスが効く|
| GLASS | 滑らかでクリーンな24 dBローパス。共鳴は控えめ |
| AMBER | 太く暖かいラダー・フィルター。軽いグリットを含む |
| WOOD | バクトロール風ローパスゲート。有機的な余韻 |

## ルーパー

spreには2種類のループシステムがあります。

### ノートルーパー

- gridの16列目、3〜8行目に6つのスロットがあります。
- 1回目の押下：録音開始。
- 2回目の押下：録音終了、再生開始。
- 以降の押下：停止／再開。
- ダブルプレス：そのスロットを消去。
- gridなしではLOOPを選択してK2を押すとスロット1を操作できます。

ノートルーパーは、AUTOで生成された音と、MIDI／GRIDモードでgridから演奏した音を記録します。外部MIDI入力は現在記録されません。演奏時のタイミングをそのまま記録し、拍にはクオンタイズされません。

### オーディオルーパー

softcutを使った2つのオーディオスロットがあります。各スロットは約30秒まで録音できます。

オーディオルーパーには、spreのエンジン出力とnornsのオーディオ入力が送られます。

- LOOPコントロールでオーディオルーパーとスロットを選びます。
- 1回目の押下：録音開始。
- 2回目の押下：次のクロック拍でループを閉じ、再生開始。
- 以降の押下：停止／再開。
- gridでダブルプレス、またはK2＋K3：選択中のスロットを消去。
- パラメータページで方向、速度（×0.25〜×4）、レベルを変更できます。

新しいオーディオ録音を終了すると、録音したループを聴き取りやすくするためFLOCKが0になります。

## 16×8 gridの配置

### AUTOモード

| grid上の位置 | 機能 |
|---|---|
| 1行目、1〜12列 | ルートの音名 |
| 1行目、15〜16列 | オクターブ・ダウン／アップ |
| 2行目、1〜16列 | スケール選択 |
| 3行目、1〜8列／9〜15列 | FLOCK／BRIT |
| 4行目、1〜8列／9〜15列 | RAIN／SAT |
| 5行目、1〜8列／9〜15列 | ATK／TAPE |
| 6行目、1〜8列／9〜15列 | DEC／DUST |
| 7行目、1〜8列／9〜15列 | FM／SPREAD |
| 8行目、1〜8列／9〜15列 | FILTER／INTONE |
| 16列目、3〜8行 | ループ・スロット1〜6。オーディオルーパーはスロット1〜2を使用 |

### MIDI／GRIDモード

| grid上の位置 | 機能 |
|---|---|
| 1行目、1〜12列 | ルートの音名 |
| 1行目、15〜16列 | オクターブ・ダウン／アップ |
| 2行目、1〜16列 | スケール選択 |
| 3〜8行目、1〜15列 | スケール音の鍵盤 |
| 16列目、3〜8行 | ループ・スロット |

gridからはMIDIモードとGRIDモードの両方で直接演奏できます。

## 収録スケール

マイナー・ペンタトニック、メジャー・ペンタトニック、メジャー、マイナー、ドリアン、リディアン、ミクソリディアン、フリジアン、ロクリアン、フリジアン・ドミナント、ブルース、ホールトーン、ディミニッシュ、ハンガリアン・マイナー、In Sen、Hirajoshi、アラビック、ビバップ・メジャー、エニグマティック、クロマチック。

norns本体の画面とMIDI CC 20からは20種類すべてを選べます。16×8 gridの2行目から直接選べるのは先頭16種類です。

## MIDI CC

| CC | 機能 |
|---:|---|
| 1〜12 | FLOCK、RAIN、ATK、DEC、FM、FILTER、BRIT、SAT、TAPE、DUST、SPREAD、INTONE |
| 13／14 | 値が63より大きいとき、オクターブ・ダウン／アップ |
| 15／16 | SUSTAIN／RELEASE |
| 17 | フィルタータイプ |
| 18 | AUTO／MIDI／GRIDモード |
| 19 | ルート。MIDIノート48〜72 |
| 20 | スケール |
| 21 | オクターブシフト。−3〜＋3 |
| 22 | メロディ生成方式 |
| 23 | ノート／オーディオルーパー選択 |
| 24〜29 | 値が63より大きいとき、ループ・スロット1〜6を操作 |
| 30／31 | 選択中のオーディオループの速度／レベル |
| 32 | softcutのpre-level |
| 33 | 選択中のオーディオループの方向。リバース／フォワード |

spreが生成したMIDIノートは、MIDIポート1のチャンネル1からも出力されます。
